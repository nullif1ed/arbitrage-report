Columns include: \texttt{time}, \texttt{binance}, \texttt{coinbase}, \texttt{spread}, \texttt{spread\_z},

\section{Very Rough Draft: Discussion \& Next Steps}
\begin{itemize}
  \item The synthetic example demonstrates the full analysis pipeline: preprocessing, spread computation, z-score signal generation, backtest, and visualization.
  \item Next: run the pipeline with real tick data (Binance trades, Coinbase trades), recompute spread at higher frequency (seconds or ms), and repeat stability analysis from our DDE model to estimate critical delay $\tau_c$.
  \item Incorporate realistic transaction costs (maker/taker), exchange transfer times, and order book depth simulation.
  \item Produce a stability map comparing theoretical $\tau_c$ to empirical PnL decay as a function of latency.
\end{itemize}
